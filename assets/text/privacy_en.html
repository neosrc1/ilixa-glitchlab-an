Ilixa Privacy Policy

Summary

Ilixa apps may collect data on how the user interacts with an app.
This is never a necessity. You have the option to opt-out of this data collection in the app's settings.
The collected data is anonymous and used to see how the apps are used in order to improve them. The collected data is not shared with any third parties except for the company that provide the analytics software, in this case Google. Here is Google's statement on this sharing of data with them:
Data you collect, process, and store using Google Analytics ("Google Analytics data") is secure and kept confidential. This data is used to maintain and protect the Google Analytics service, to perform system critical operations, and in rare exceptions for legal reasons as described in our privacy policy.


What we collect

Here is a sample of the data we collect:
  1. location
  2. device type
  3. Android Advertising ID
  4. when and what type of events occured (for example selecting an effect, saving an image etc.)
  5. stack traces (where in the app's code a crash occured)


Where we store the data

We use third party analytics packages to help us collect data:
  1. Google Analytics for Firebase for data that relates to how the apps are used.
  2. Crashlytics for data that relates to apps misbehaving.


How we use the information we collect

Ilixa uses the collected information exclusively to improve its apps:
  1. for example understanding what features of an app are the most used allows us to focus more on those features.
  2. having crash reports makes it easier to identify bugs in the apps.
